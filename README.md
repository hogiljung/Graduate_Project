# Graduate_Project
# 개요

## 개발 배경
당구는 2018년 기준 전국의 당구장 수가 PC방 수를 넘을 만큼 인기 있는 스포츠이나,
당구대, 큐 등의 장비로 인한 물리적 제약이 존재한다.

## 개발 목표
1. VR 환경 내 당구공의 움직임 구현
2. 리플레이 기능
3. 성공을 위한 타법 제안(가이드)

## 개발 효과
당구의 접근성 확대
